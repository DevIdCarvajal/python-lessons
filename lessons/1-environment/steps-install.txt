
Persona 1:
  1. Crear un entorno virtual
  2. Crear un repositorio Git y enlazarlo con GH
  3. Trabajar en el proyecto (código y paquetes)
  4. Crear requirements.txt
  5. Subir el trabajo realizado (sólo código y el requirements.txt, no los paquetes) a GH

Persona 2:
  1. Crear un entorno virtual
  2. Clonar el proyecto de GH
  3. Instalar todos los paquetes y sus versiones especificadas en el fichero requirements.txt
